Automated Predictive Maintenance Pipeline for Air Pressure System (APS) - Sensor Fault Detection

Collaborated with a team to build a fully automated pipeline for predicting failures in heavy-duty vehicle APS, aiming to reduce repair costs and improve uptime.

Challenge: Addressed a binary classification problem to differentiate between APS component failures and other vehicle issues, focusing on minimizing false positives.

Solution:

  • Developed a data ingestion process using Apache Kafka to stream real-time sensor data to MongoDB.
  • Implemented data validation and transformation to ensure data quality for analysis.
  • Designed and trained machine learning models (Random Forest, XGBoost) for accurate predictions.
  • Established a model evaluation process using precision, recall, and F1-score metrics.
  • Created a model pusher for automated deployment on EC2.
  • Containerized the pipeline with Docker for scalable deployment across various environments.
